Objects

In computer science, an object can be a variable, a data structure, a function, or a method. As regions of memory, they contain value and are referenced by identifiers. They can be Lists or Dictionaries in Python.

Simply put, an object is an instance of a class below.

A direct way to create objects is what we call an 'object literal'. We just hard code the values in. 

        numsList = [1, 3, [2, 4, 6]]
        daysDictionary = { 'Sun', 'Mon', 'Tue', 'Wed' }
        emptyObjectList = []
        emptyObjectDictionary = {}

An object can also be instantiated or created by the following "constructor" function or Class and finally is saved to the variable, myHero, which takes on the attributes and methods.

        class RangedAssaassin:
            #Attributes
            strength = 100
            health = 100

            #methods
            def sayHi(self):
                return 'hi'
    
        #Instantiation
        myHero = RangedAssaassin()
