Classes

In object-oriented programming, a class is an extensible program-code-template or blueprint for creating objects, providing initial values for state (member variables or attributes) and implementations of behavior (member functions or methods).

An example of a class in Python code would be a fictional character type for an online video game. 

        class RangedAssaassin:
            #'__init__' is the initial state class 
            def__init__(self, heroName): 
                #Base attributes after making an assassin obj 
                self.heroName = heroName
                self.attackType = "ranged"
                self.health = 1340
                self.healthReagen = 2.7
                self.attackRadius = 0.625
                self.attackRange = 5.5
                self.speed = 1.67
                self.attackDamage = 70
    
                #Base methods you can call on after making assassin obj
                def sprint(self, charge):
                    self.speed = self.speed * charge
                def furtherReach(self, vision):
                    self.attackRange = self.attackRange * charge
